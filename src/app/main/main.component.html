Online Shop

This is a fully functional online shopping web application built with Angular, where users can browse products, add them to a cart, and proceed to checkout.

Features:

  • Dynamic Product Listing: Products are fetched from a mock API and displayed in a responsive, user-friendly interface.
  • Shopping Cart: Users can add/remove items, view cart totals, and update quantities.
  • Authentication: User login functionality with cookies, ensuring that users can only access certain features once logged in.
  • State Management: Using Angular services and RxJS to handle app state, including the cart, user authentication, and other shared data.
  • Search and Filters: Users can search for products and apply filters to narrow down their selection based on categories, price range, ratings, and other attributes. The filters allow users to quickly find products that meet their needs, improving the overall shopping experience.
  • API Integration: The project integrates with a mock backend to simulate real-time product fetching, shopping cart updates, and user data management.

Hotel Booking

This is a hotel booking web application built with Angular where users can browse available rooms, check availability, and book their stay online.

Features:

  • Room Listings: Displays available hotel rooms with details such as price, amenities, and images, fetched from a backend API.
  • Search and Filters: Users can search for rooms by date, type, and other filters to find the best accommodation option.
  • Booking System: Guests can select a room, choose check-in/check-out dates, and confirm the booking.
  • User Authentication: Includes login and registration functionality for users to manage their bookings, view past reservations, and access special offers.
  • State Management: Manages data related to booking details, user information, and room availability using Angular services and RxJS for reactive state management.
  • Responsive Design: Designed to be mobile-first using css for easy navigation on any device.
  • API Integration: Integrates with a backend service for fetching room data, making bookings, and saving user information.

Online Restaurant

This is a restaurant web application built with Angular, designed to showcase the restaurant's menu, allow users to place orders, and interact with the restaurant's services.

Features:

  • Menu Display: A dynamic list of restaurant menu items, including food descriptions, prices, and images. The menu items are fetched from a backend API, allowing for easy updates.
  • Search and Filtering: Users can search for menu items by categories (e.g., appetizers, main course, desserts) or filter by dietary preferences (e.g., vegetarian, gluten-free).
  • Order Management: Customers can add items to their order, view the total price, and finalize their orders. This functionality includes a cart system where users can modify quantities or remove items.
  • User Authentication: Includes login and registration features, allowing customers to manage their orders and access personalized offers.
  • State Management: Uses Angular services and RxJS for state management, keeping track of the user’s cart, order details, and user authentication status.
  • Responsive Design: A fully responsive design built with css, ensuring the app works well on any device from desktop to mobile.
  • API Integration: Integrated with a backend service to fetch menu data, submit orders, and handle customer authentication.
